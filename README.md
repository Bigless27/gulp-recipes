# gulp-recipes

Collection of gulp recipes

## Quick Start

### Pre-requisites

```bash
npm install -g gulp
```

### Install

```bash
git clone https://github.com/sogko/gulp-recipes
cd gulp-recipes

npm  install

```

### Running a recipe example

For eg: [```expressjs-browser-sync-nodemon```](expressjs-browser-sync-nodemon)

```bash
cd expressjs-browser-sync-nodemon

gulp
```

# gulp-recipes

Collection of gulp recipes for most common tasks

# Quick Start

## Pre-requisites

```
npm install -g gulp

npm install -g bower

```

## Install

```bash
git clone https://github.com/sogko/gulp-recipes
cd gulp-recipes

npm  install

```

## Running a recipe example

For eg: ``expressjs-browser-sync-nodemon```

```bash
cd expressjs-browser-sync-nodemon

gulp

```
